# Cardapio_QR
Focado em eliminar a dor de pequenos restaurantes com cardápios de papel.  O sistema oferece um painel admin com CRUD completo (Categorias, Itens) para o gerenciamento fácil e instantâneo do menu. Gera um QR Code para acesso via smartphone. Foco em redução de custos, modernidade e facilidade de atualização, ideal para validar um produto de nicho.
